# Configuration

BraceKit provides comprehensive settings to customize your AI assistant experience. Access settings by clicking the **Settings** icon (⚙️) in the header.

## Settings Tabs

| Tab | Description |
|-----|-------------|
| [**AI**](./ai-provider/) | Configure AI providers, API keys, and model parameters |
| [**Chat**](./chat/) | Customize chat behavior, system prompt, and text selection |
| [**Compact**](./compact/) | Configure auto-compact settings for long conversations |
| [**Memory**](./memory/) | Manage persistent memory and personalization |
| [**MCP**](./mcp-servers/) | Connect external tools via MCP servers |
| [**Data**](./data/) | Export, import, and backup your data |
| [**Safety**](./safety/) | Password protection and security settings |
